# Дизайн-система

## Направление

UX-структура — как у **Аскона** (лидер по юзабилити среди мебельных РФ): витрина с акцентом на каталог, насыщенные карточки товара, сильный поиск, готовые комплекты, шопабельные интерьеры, статус наличия, услуги в карточке и корзине. Мы повторяем эти **паттерны и структуру**, но с **собственной визуальной идентичностью** (палитра + шрифты), а не ассеты Аскона.

Аудитория: покупатель мебели среднего сегмента в РФ. Задача страницы: быстро найти и купить мебель. **Сигнатурный элемент** — поиск по фото (он же выносится в герой). Тон — спокойный, премиальный, без визуального шума. Смелость тратим в одном месте (поиск по фото); всё вокруг — тихо и дисциплинированно.

## Токены цвета

Дефолтная палитра (тёплый нейтральный фон + чернильно-синий акцент + тёплый янтарь для распродаж). Сознательно НЕ бирюза Аскона и НЕ клишированный «крем + терракота».

```css
:root {
  /* поверхности */
  --bg: #FAF9F7;          /* тёплый почти-белый фон */
  --surface: #FFFFFF;     /* карточки */
  --surface-2: #F1EFEA;   /* секции-разделители */
  --line: #E6E2DB;        /* границы */

  /* текст */
  --ink: #1F1D1A;         /* основной (тёплый near-black) */
  --ink-muted: #736E66;   /* вторичный */

  /* бренд / действия */
  --accent: #23395B;      /* primary: кнопки, ссылки, бренд (глубокий чернильно-синий) */
  --accent-hover: #1A2C47;
  --accent-ink: #FFFFFF;  /* текст на акценте */

  /* акценты состояний */
  --highlight: #C8893F;   /* распродажа, рейтинг — ИСПОЛЬЗОВАТЬ СКУПО */
  --success: #2E7D52;     /* в наличии */
  --danger: #B23B3B;      /* ошибки */

  /* форма */
  --radius-card: 12px;
  --radius-control: 8px;
  --shadow-1: 0 1px 2px rgba(31,29,26,.06), 0 2px 8px rgba(31,29,26,.04);
  --shadow-2: 0 8px 28px rgba(31,29,26,.10);

  /* сетка */
  --space: 4px;           /* шаг spacing-шкалы (×1,×2,×3,×4,×6,×8,×12,×16) */
  --container: 1280px;
}
```

## Перекраска под клиента (ключевой рычаг)

Это шаблон на много клиентов: **ребрендинг = смена токенов**, код не трогается. На клиента меняем обычно две вещи:

1. `--accent` (+ `--accent-hover`) — фирменный цвет.
2. (опц.) `--bg` / `--surface-2` — тёплый/холодный/тёмный нейтральный.

Токены живут в `site.config.ts` тенанта и инжектятся в `:root`. Семантические имена (`--accent`, `--success`), а не «синий», — поэтому смена палитры не ломает смысл.

## Типографика

Обе гарнитуры — нативная кириллица, бесплатные (Google Fonts):

- **Display — `Onest`** (600–800): заголовки, герой. Современный, чуть характерный, не дефолтный Inter. Используется крупно и сдержанно.
- **Body / UI — `Golos Text`** (400–600): текст, интерфейс. Сделан под русские интерфейсы, отличная читаемость.
- **Цены и спецификации:** Golos Text + `font-variant-numeric: tabular-nums` (цифры не «прыгают»).

Шкала (clamp для адаптива): H1 `clamp(2rem, 5vw, 3.25rem)` / H2 `2rem` / H3 `1.375rem` / body `1rem` / small `.875rem` / caption `.75rem`. Межстрочный: заголовки 1.1, текст 1.55.

## Компоненты

- **Header (sticky):** лого · мега-меню каталога · строка поиска (иконка камеры = поиск по фото + автоподсказки) · город/шоурум · корзина · телефон.
- **ProductCard:** фото (hover = 2-е фото) · название · цена + старая цена + бейдж скидки (`--highlight`) · бейдж наличия (`--success`) · быстрое «в корзину».
- **Кнопки:** primary (`--accent`), secondary (контур), ghost. Радиус `--radius-control`. Текст действия = что произойдёт («В корзину», «Оформить заказ»).
- **Бейджи:** наличие, скидка, «хит», «новинка».
- **Фильтры:** сайдбар, чекбоксы/диапазоны, авто из `attributes`; мультивыбор; чипы активных фильтров.
- **Photo-search dropzone:** зона drag&drop + кнопка загрузки + состояние «анализирую…» (см. `PHOTO-SEARCH.md`).

## Контент (копирайтинг интерфейса)

- Активный залог, что делает кнопка: «В корзину», «Оформить заказ», «Показать похожие».
- Одно действие — одно имя на всём пути (кнопка «Оформить заказ» → тост «Заказ отправлен»).
- Пустые состояния — приглашение к действию («Загрузите фото — подберём похожую мебель»), а не настроение.
- Ошибки — что случилось и как исправить, без извинений и воды.

## Quality floor (без оговорок)

- Адаптив до мобильного (фильтры → шторка снизу; герой и гриды перестраиваются в 1–2 колонки).
- Видимый фокус с клавиатуры, корректные `alt`, контраст AA.
- `prefers-reduced-motion` уважается; анимации сдержанные (hover карточек, мягкие переходы, без «AI-шлейфа»).
- `tabular-nums` для всех цен.
