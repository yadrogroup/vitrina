# Vitrina — шаблон сайта мебельного магазина

Продуктизируемый шаблон интернет-магазина мебели: один кодовая база → много клиентских сайтов. Каждый сайт — **статика** (Astro SSG), которая хостится на домене клиента. Фишка — **поиск по фото** («загрузи фото → похожие товары»), работающий **целиком в браузере, без сервера**.

Модель та же, что в шаблоне для стоматологий: общий код, per-client конфиг (тема + контакты + источник каталога), сборка через Cursor AI.

## Стек

- **Astro** (SSG, острова интерактивности) + TypeScript
- **Поиск по фото:** transformers.js / ONNX Runtime Web + квантованный CLIP (в браузере)
- **Тексты/SEO:** GigaChat (только на сборке)
- **Заказы:** один общий serverless-релай (scale-to-zero) → email + Telegram клиента
- **Хостинг:** статика на домене клиента (или дешёвый Object Storage)
- **Стили:** CSS-переменные (design tokens), без тяжёлых UI-фреймворков

## Принцип: нет постоянного backend → нет подписки

Сайт статический, поиск по фото считается в браузере, GigaChat работает разово на сборке. Единственная серверная мелочь — общий релай заказов (почти бесплатно, один на всех клиентов). Поэтому сайт **продаётся разово**, без подписки.

## Быстрый старт

```bash
npm install
cp .env.example .env        # заполнить: источник каталога, ключи, контакты
npm run sync                # синк каталога из источника → data/catalog.json
npm run embed               # посчитать эмбеддинги картинок → data/embeddings.json
npm run enrich              # (опц.) GigaChat: тексты карточек + SEO
npm run client:build        # полный пайплайн: sync → embed → enrich → build
npm run verify-theme        # accent из site.config.ts попал в dist, нет лишних hex
npm run dev                 # локальная разработка
npm run build               # статика в ./dist → деплой на домен клиента
```

Подробный чек-лист клиента — [`ONBOARDING.md`](ONBOARDING.md).

## Per-client конфигурация

Один клиент = один файл `site.config.ts` (название, палитра, контакты, Telegram chat id, источник каталога) + его веса CLIP в `public/models/`. Меняешь конфиг → пересобираешь → деплоишь. Код не трогаешь.
